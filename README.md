# GymTracer_Frontend
Frontend oldalak

## 1. Regisztráció és Bejelentkezés
- Regisztráció
- Bejelentkezés

## 2. Bejelentkezés nélküli felület
- Kezdőlap: Általános infók, Jegyárak, Edzések

## 3. User felület
- Profil: adatok/módisítás
- Kártyáim: kérése/mutatása
- Jegyeim: vásárlás, megnézés, fizetés
- Edzések: naptárszerűen, jelentkezés, keresés, lemondás

## 4. Trainer felület
- User + Trainer
- Saját trainingek: létrehozása, törlés, megjelentek bejelölése

## 5. Staff felület
- User + Trainer + Staff(váltó gomb Staff felületre)
- Előző funkciók más nevében
- User kiválasztó menü (QR code beolvasással is)
- Statisztika (2 grafikon)

## 6. Admin 
- User + Trainer + Staff(váltó gomb Staff felületre + admin funkciók)
- Jogosultságok módosítása
- Bevétali statisztikák
- Kártyahasználati statisztikák
